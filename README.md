# Introduction to Bibliographic Data Science
A training material

Implementation of the essay _An outline of an imagined training course on bibliographic data science_
[Bibliographic data](https://bibliodata.substack.com/p/an-outline-of-an-imagined-training) blog, 2026.

The book is written in Markdown files, and rendered with [Quarto](https://quarto.org). To run it locally:

```bash
quarto render
```

It will create the HTML pages in the directory `_book`.

Commits at Github launch a deployment process that publishes the textbook to https://pkiraly.github.io/introduction-to-bibliographic-data-science/.
